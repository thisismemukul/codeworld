Tricks

Commonn CSS mistakes

Using px when its not needed

Write only 0 in place of 0px.


.class{   X
padding:16px 0px;
}
.class{   
padding:16px 0;
}

Not using shorthand properties

Save time by using this trick.


.class{   X
margin-top:20px;
margin-bottom:20px;
margin-left:20px;
margin-right:20px;
}
.class{   
margin:20px;
}

No font fallback

Write more than one font.


.class{   X
font-family:Bebas Neue;
}
.class{   
font-family:Bebas Neue, Arial, sans-serif;
}
