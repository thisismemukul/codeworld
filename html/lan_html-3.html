Using CSS in HTML

CSS can be addded to HTML doc in three main ways:

  • Inline:

    by using style attribute in HTML elements.

  • Internal:

    by using <style>element in <head>section.

  • External:

    by using a <link>element to link to an external CSS file.

Inline CSS

An inline CSS uses <style>element in <head>section.

  • Lets take an example.
  • In this example we are going to set the text colour of <h2> element to green, and text colour of <p> element to red..

    • <!doctype html>
    <HTML>
    <HEAD>
    <TITLE>Welcome to Code World </TITLE>
    </HEAD>
    <BODY>
    <h6 style="color: green;">
    This is heading with green colour. </h6>
    <p style="color: red;">
    CODE WORLD is awesome. </p>
    </BODY>
    </HTML>
  • Output:
    • This is heading with green colour. CODE WORLD is awesome.

Internal CSS

An internal CSS is used to define a style for a single HTML page.

An internal CSS is defined in the <head>section of an HTML page,within a <style>element.

  • Lets take an example.
  • In this example we are going to set a internal CSS that changes heading colour to green and text colour of paragraph to red.

    • <!doctype html>
    <HTML>
    <HEAD>
    <TITLE>Welcome to Code World </TITLE>
    <style>
    h6 {color: green;}
    p {color: red;}
    </style>
    </HEAD>
    <BODY>
    <h6> This is heading with green colour. </h6>
    <p>CODE WORLD is awesome. </p>
    </BODY>
    </HTML>
  • Output:
    • This is heading with green colour. CODE WORLD is awesome.

External CSS

An external CSS is used to define a style for more than single HTML page.

To use an external CSS we need to add a link in the <head>section of each HTML page by using <link>element.

  • Lets take an example.
  • In this example we are going to add a link for external CSS that changes heading colour to green and text colour of paragraph to red.

    • <!doctype html>
    <HTML>
    <HEAD>
    <TITLE>Welcome to Code World </TITLE>
    <link rel="stylesheet" href="style.css">
    </HEAD>
    <BODY>
    <h6> This is heading with green colour. </h6>
    <p>CODE WORLD is awesome. </p>
    </BODY>
    </HTML>

  • The external css must saved with a .css extension,and link should be carefully written in html doc.

    • Below is the "style.css" file

    "style.css"

    body {
    background-color : transparent ;
    }
    h1 {
    color : green ;
    }
    p {
    color : red ;
    }

  • Output:
    • This is heading with green colour. CODE WORLD is awesome.
