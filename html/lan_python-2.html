Python Installation

How to install Python?

  • Many PCs and Macs will have python already installed.
  • To check if you have python installed on a Windows PC,run the following on the Command Line.
    • C:\Users\user-name>python --version

  • If not installed download it.
  • Latest version of python is python 3.8 you can download it from its official site.

The Python Command Line

  • Type the following on the command line:
    • C:\Users\user-name>python
    Or, if the "python" command did not work, you can try "py":
    C:\Users\user-name>py
  • After this you will see the following lines on your command line..
    • C:\Users\user-name>python
    Python 2.7.18 (v2.7.18:8d21aa21f2, Apr 20 2020, 13:19:08) [MSC v.1500 32 bit (Intel)] on win32 Type "help", "copyright", "credits" or "license" for more information.
    >>>

Lets write a Hello World program in Python.

  • Hello World
    • >>> print("Hello World!")
    Output:
    Hello World!

    Creating a Comment line in Python

  • Comments starts with a #, and Python will ignore them.
    • #This is a comment
    #Python will ignore these lines
    >>> print("Hello World!")
    Hello, World!
